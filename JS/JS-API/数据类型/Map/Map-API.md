# Map-API

## 原型属性

- size

## 原型方法

- clear()
  清除 Map 中所有的键值对

- delete(key)
  删除指定键值对, 返回 Boolean

- get(key)
  返回指定 key 对应的值, 没有则返回 undefined

- has(key)
  查询是否存在指定 key, 返回 Boolean

- set(key, value)
  设置键值对, 返回 Map 对象

- keys()
  返回一个新的迭代器, 包含所有的 key 数组

- values()
  返回一个新的迭代器, 包含所有的 value 数组

- entries()
  返回一个新的迭代器, 包含 `[key, value]` 格式的数组

- forEach((value, key, map) => {})
