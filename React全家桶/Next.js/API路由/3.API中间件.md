# API 中间件

API 路由提供了内置的中间件

**用于解析传入的请求.**

- `req.cookies` 一个包含 cookie 的对象
- `req.query` 一个查询字符串对象
- `req.body`

针对 Connect/Express 中间件的支持

可以利用 cors 软件包为 API 端点配置 CORS

**用于处理响应的帮助函数**

- `res.status(code)` 用于设置响应状态码
- `res.json(body)` 发送一个 json 数据
- `res.send(body)` 发送任意数据
- `res.redirect(path)` 重定向
