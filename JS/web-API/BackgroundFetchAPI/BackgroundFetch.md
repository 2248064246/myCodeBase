
# BackgroundFetch

后台 Fetch API 提供了一个基于 Service Worker 的下载和上传机制，该机制在 Service Worker 和浏览器重启时是持久的。系统将提供一个开发者可控的界面来调度、查询、暂停和取消后台下载和上传。预定获取完成后，系统将通知开发人员操作的结果。正在进行的提取将在通知中对用户可见。
