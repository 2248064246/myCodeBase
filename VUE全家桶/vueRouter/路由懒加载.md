
# 路由懒加载
> 就是在进入这个路由的时候才去加载这个组件

> 如果路由很多, 避免一个路由文件过大, 防止加载变慢
 

## 使用 webpack 动态 import() 语法
> import() 会返回一个 promise

> babel 解析需要 ` syntax-dynamic-import` 插件

```javaScript
const router = new VueRouter({
  routes: [{ path: '/foo', component: () => import('xxx') }]
})

```
