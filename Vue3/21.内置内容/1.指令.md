## v-pre

跳过该元素及其所有子元素的编译。

元素内具有 v-pre，所有 Vue 模板语法都会被保留并按原样渲染。

## v-once

仅渲染元素和组件一次，并跳过之后的更新。

## v-cloak

用于隐藏尚未完成编译的 DOM 模板。

v-cloak 会保留在所绑定的元素上，直到相关组件实例被挂载后才移除。配合像 [v-cloak] { display: none } 这样的 CSS 规则，它可以在组件编译完毕前隐藏原始模板。
