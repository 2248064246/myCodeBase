# linux 下安装jupyter

## 安装 python3 和 pip3

ubuntu 或 deepin 系统中, 自带 python2

```
sudo apt-get install python3
```

一句话搞定


检查是否安装正确

```
python3 --version

pip3 --version
```

> 如果没有默认自带 pip3, 需要手动安装

```
sudo apt-get install python3-pip

sudo pip3 install --upgrade pip
```
