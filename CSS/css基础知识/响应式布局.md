## 响应式布局
+ 网页可以根据不同的设备或窗口大小呈现不同的效果
+ 使用响应式布局,可以使一个网页使用于所有设备
+ 响应式布局的关键就是 **媒体查询**
+ 通过媒体查询,可以为不同设备,或设备不同状态来设置不同样式

```css
    /* 使用媒体查询 
    *       语法: @media 查询规则{}
                all 所有设备
                print 打印设备
                screen 带屏幕的设备
                speech 屏幕阅读器
                - 可以使用 , 连接多个媒体查询

            可以在媒体类型前添加一个only, 表示只有
                主要是为了兼容老一些的浏览器
    */
    @media only all{

    }

    /* 
        媒体特性:
            width 视口的宽度
            height 视口的高度

            min-width 视口的最小宽度 (视口大于指定宽度时生效)
            max-width 视口的最大宽度 (视口小于指定宽度时生效)
            
            样式的切换分界点, 我们称为断点, 也就是网页样式会在这个点发生变化
            一般比较常用的断点
                小于768 超小型屏幕 max-width = 768px
                大于768 小屏幕 min-width = 768px
                小于992 中型屏幕 max-width=992px
                大于1200 大型屏幕 min-width= 1200px 
     */
     @media (min-width: 500px) {

     }
     @media only screen and (min-width: 500px) and (max-width: 700px) {
         /* 大于500 小于 700 */

         /* , and not (或 与 非) */
     }
```
