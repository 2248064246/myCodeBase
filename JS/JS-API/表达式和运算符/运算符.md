# 运算符

## 赋值运算符

- **赋值 `=`**
- **加法赋值 `+=`**
- **减法赋值 `-=`**
- **乘法赋值 `*=`**
- **除法赋值 `/=`**
- **求余赋值 `%=`**
- **求幂赋值 \*\*=**
- **左移位赋值 `<<=`**
- **右移位赋值 `>>=`**
- **按位或赋值 `|=`**
- **按位与赋值 `&=`**
- **按位异或赋值 `^=`**
- **逻辑空赋值 `??=`**
- **逻辑和赋值 `&&=`**
- **逻辑或赋值 `||=`**

## 算数运算符

- `+ - * / %`
- `**` 指数运算符

## 位运算符

- `&`
- `|`
- `~`
- `^`
- 右移位 `>>`
- 左移位 `<<`
- 无符号右移位 `>>>`

## 逻辑运算符

- `&&` 逻辑和
- `||` 逻辑或
- `??` 逻辑空
- `!`

## 字符串运算符

- `+`

## 条件(三元)运算符

- `? :`

## 逗号运算符

- `,`

## 一元运算符

- `++`
- `--`
- `+` 一元加运算
- `-` 一元减运算
- `~` 按位非
- `!` 逻辑非

- `<`
- `==`
- `===`

- `<=`
- `!=`
