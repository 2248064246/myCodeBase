# Set API

## 静态属性

- size  
  返回 Set 中对象个数

## 实例方法

- add(value)
  在 Set 尾部添加一个对象, 返回该 Set 对象

- clear()
  移除 Set 对象内所有元素

- delete(value)
  删除指定值, 返回一个 Boolean 值

- has(value)  
  判断 Set 中是否存在某个值

- keys() | values()
  这个方法相同, 都是按照插入顺序返回集合中的所有元素

- forEach()

- entries()
  返回一个新的迭代器, 包含所有按照插入顺序的所有元素的值的数据 `[value, value]` 形式
