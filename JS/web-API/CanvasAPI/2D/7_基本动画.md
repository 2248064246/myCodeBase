# 基本动画

## 动画的基本步骤

1. 清空 canvas
   - 除非接下来要画的内容会完全充满 canvas （例如背景图），否则你需要清空所有。最简单的做法就是用 clearRect 方法。
2. 绘制动画图形 
   - 重绘动画帧
3. 恢复canvas状态
   - 如果已经保存了canvas的状态, 可以先恢复它, 然后重绘下一帧

## 操控动画

这个指的控制动画帧的绘制, 这个通常由 `setInterval` `setTimeout` 完成

现代浏览器可以通过 `requestAnimationFrame` 来定期绘制(在重绘之前，请求浏览器执行一个特定的函数来更新动画。)
