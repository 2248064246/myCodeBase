# 直线与方程

## 倾斜角与斜率

斜率: `k=tan α`

直线斜率公式: `k = (y2-y1)/(x2-x1)`

## 两条直线平行与垂直的判定

1. 平行: `k1=k2`
2. 垂直: `k1*k2=-1`

## 直线的方程

- 斜截式 `y=kx + b`
- 点斜式 `y-y0 = K(x - x0)`
- 两点式 `(y-y1)/(y2-y1) = (x-x1)(x2-x1)
- 一般式 `Ax + By + c = 0`

## 直线的交点坐标与距离公式

两条直线的交点坐标就是两条直线方程的解

## 两点距离

`|P1P2| = √((x2-x1)^2 + (y2-y1)^2)`

## 点到直线的距离

`d = (|Ax0 + By0 + C|)/ √(A^2 + B^2)` 点 P(x0, y0) 到直线 `Ax + By + C = 0` 的距离

## 两条平行线的距离

> 两条平行直线间的距离是指夹在两条平行直线将公垂线段的长度
