
# JS 一些小知识

## ~ 运算
```javascript
  ~2021 // => -2022
  ~-1 // => 0
  ~0 // => -1
```

## ~~ 运算
```javascript
  ~~1.123 // => 1
  ~~1.89  // => 1
```

## + 运算
```javascript
  +'1' // => 数字 1
  +new Date() // => 等同于 Date.now(), 获取当前时间戳
```

## += 和 ++
> += 和 ++ 并不一样, ++ 是严格的数学运算
```javascript
  var a = '1'
  a+= 1 // => a='11'
  // ==============
  a++ // => a=2
``
