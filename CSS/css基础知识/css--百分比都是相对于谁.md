# CSS--百分比都是相对于谁计算
[Toc]
### 相对于父级元素宽高
+ `width`, `height`, `margin`,`padding`, `text-indent` ... 等

### 相对于父级定位元素宽高
+ `top`, `left`, `right`, `bottom`

### 相对于自身宽高
+ `border-radius`, `translate`, `transform-origin`, `zoom,background-size`...

### 相对于自身字体大小
+ `line-height`

### 相对于继承字体大小
+ `font-size`
